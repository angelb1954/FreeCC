Iterar con bucles while de JavaScript

Puede ejecutar el mismo código varias veces mediante un bucle.

El primer tipo de bucle que aprenderemos se llama whilebucle porque se ejecuta mientras una condición específica es verdadera y se detiene una vez que esa condición ya no es verdadera.

const ourArray = [];
let i = 0;

while (i < 5) {
  ourArray.push(i);
  i++;
}
En el ejemplo de código anterior, el whilebucle se ejecutará 5 veces y agregará los números del 0 al 4 a ourArray.
